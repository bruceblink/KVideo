# Changelog

KVideo 使用语义化版本记录可交付版本。尚未发布的改动放在 `Unreleased`，发布时再移动到对应版本。

## Unreleased

- 明确云同步服务与用户自有内容来源的产品边界。
- 制定 SyncHub 付费同步服务的分阶段开发计划。
- 将当前产品主线调整为播放器可靠性、故障恢复和跨端体验，并为同步服务商业化设置质量与用户需求门槛。

## 0.1.0 - 2026-07-18

- 以独立仓库和 AGPL-3.0 许可证开始维护 KVideo。
- 提供多源搜索、HLS 播放、收藏、观看历史、账户隔离与自托管部署能力。
- 接入 SyncHub，通过 KVideo 应用 API Key 同步观看历史和收藏。
